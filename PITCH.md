# Kingdom Web

## What is this?
Web-based hybrid kingdom management / first-person grid dungeon crawler.
A design lab for Kingdom Crawler — explore game systems with more depth
in a browser, using programmatic art instead of pixel art.

## Why?
Parallel exploration of the Kingdom Crawler design space with different
constraints. This is a lab, not a product — bias toward trying things.
Lessons learned feed back into the Godot version.

## Vibe
Programmatic vector art. Zero external art assets — all visuals from code
(SVG, Canvas 2D, shaders). Systems-deep, easy to pick up after weeks away.

## Tech
TypeScript, web browser, programmatic art. Same patterns as hexcrawler
and text-adventure-engine for returnability.
